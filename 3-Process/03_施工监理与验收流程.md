# 文物保护工程施工、监理与验收流程

> 本文档梳理文物保护工程 **实施阶段（施工 → 监理 → 检查 → 验收 → 归档）** 的完整流程与要求。
> 依据：《文物保护工程管理办法》第十七～二十七条、《检查管理办法》、《竣工验收管理办法》、《文物保发〔2025〕41号》。

---

## 一、招投标与开工

### 1. 招投标范围
- **修缮工程、保护性设施建设工程、迁移工程** 实行 **招投标和工程监理**。
- 重要工程按分级报批 **招标文件** 及拟选施工单位。
- 施工、监理单位须具备相应 **文物保护工程资质**。

### 2. 开工报告（2025年41号新增要求）
- 技术监督部门通过 **开工报告** 启动全流程监管。
- 施工前对参与人员进行 **文物保护法律法规、安全责任培训**。
- 施工现场设立 **文物保护警示标识**。
- 建立 **施工、监理单位主要管理人员在岗备案制度**。

---

## 二、施工实施（《管理办法》第十九条）

### 1. 施工单位工作程序
1. 依据设计文件编制 **施工方案**（按《文物建筑保护工程施工组织设计编制要求》）；
2. 施工人员进场前接受文物保护知识培训；
3. 做好施工记录和施工统计文件，收集文物资料；
4. 进行质量自检，**隐蔽部分必须与业主、设计、监理单位共同检验并记录**；
5. 提交竣工资料；
6. **保修期自竣工验收之日起计算，不少于五年**（保养维护、抢险加固除外）。

### 2. 施工程序管控（2025年41号通知，⭐流程节点）

严格执行以下程序，及时规范形成工程资料：
```
现场复核 → 补充勘察 → 工程洽商 → 施工记录 → 阶段验收
```

### 3. 影像与数字化记录（强制）
- **构件登记、关键工序、隐蔽工程、试验样板必须做好影像记录**；
- 鼓励开展 **全过程工程摄影和数字化记录**。

### 4. 现场技术管理要求
- 严格控制施工范围和干预力度，坚持传统材料和工艺；
- 逐个做好 **梁架、斗栱、柱** 等关键结构构件、屋顶构造节点及装饰构件的修缮做法论证和记录；
- 加强材料制备试验检验：木材含水率、灰土、瓦件、砖石检测；
- 砖石构件风化、断裂但不影响结构和使用安全的，**原则上保留续用**；
- **古建筑修缮施工区域严禁使用明火；近现代建筑修缮不得使用防水卷材热熔法**。

---

## 三、施工中的特殊情况处理

### 1. 发现新文物（《管理办法》第二十条）
- 施工中发现新的文物、有关资料或影响文物保护的重大问题：
  1. **立即记录**；
  2. **保护现场**；
  3. 经 **原申报机关向原审批机关报告**，请示处理办法。

### 2. 设计变更（《管理办法》第二十一条，⭐分级管控）
| 变更类型 | 处理程序 |
|----------|----------|
| 变更/补充 **技术设计** | 业主、设计、施工共同现场洽商，**报原申报机关备案** |
| 变更 **工程项目** 或方案设计 **重要内容** | **经原申报机关报原审批机关批准** |

> ⚠️ 2025年41号通知：**严查以工程洽商替代设计变更的行为**。系统应强校验，区分"工程洽商"与"设计变更"。

---

## 四、工程监理

- 修缮、保护性设施建设、迁移工程 **实行工程监理**。
- 监理单位须具备 **文物保护工程监理资质**，配备 **责任监理师**。
  - **甲级**：可监理所有级别工程；
  - **乙级**：可监理二级及以下工程。
- 监理内容：质量、进度、安全、文物保护原则落实、隐蔽工程验收、设计变更审核等。

---

## 五、工程检查（事中监管）

依据《文物保护工程检查管理办法（试行）》：
- 检查主体：各级文物行政部门 + 技术监督部门；
- 检查内容：工程程序、质量、效果、安全；
- 被检查业主单位应 **协调督促勘察设计、施工、监理单位及时整改**；
- 检查与整改文件立卷存档。

### 2025年41号通知的技术监督全流程：
```
开工报告 → 关键节点监督 → 材料检验检测 → 工地检查
→ 竣工报告审核 → 验收 → 监管记录生成 → 竣工档案保存
→ 提出责令整改或停工建议
```

---

## 六、竣工验收

### 1. 竣工验收程序
```
施工单位自检合格
      ↓
业主单位组织四方验评（设计/施工/监理/业主）+ 编制竣工报告
      ↓
原申报机关初验合格
      ↓
审批机关（竣工验收部门）成立验收小组 → 组织竣工验收
      ↓
出具正式竣工验收结论（不得以专家意见/四方意见代替）
      ↓
竣工档案抄报相应级别技术监督部门
```

### 2. 竣工资料清单
- 工程总结报告、竣工报告；
- 竣工图纸；
- 财务决算书及说明；
- 勘察设计、施工、监理单位的总结与质量评价；
- 施工记录、隐蔽工程验收记录、材料试验报告；
- 工程档案（按《文物保护工程文件归档整理规范》）；
- **修缮前后各建筑单体各方向同角度对比照片**（2025年41号新增）。

### 3. 验收主要内容
1. 程序合规性；2. 设计落实情况；3. 工程质量；4. 文物保护原则落实；5. 资料完整性；6. 安全与环保。

### 4. 分阶段验收
- 按工序 **分阶段验收**；
- 重大工程告一段落时，审批机关组织 **阶段验收**。

---

## 七、保修与工程报告出版

- **保修期≥5年**（自竣工验收之日起，保养/抢险除外）；
- 全国重点文物保护单位 **重点项目**：验收后 **三年内发表工程报告**；
- 鼓励拍摄修缮专题纪录片、设置专题展陈；
- 具备条件的可在隐蔽位置设 **小型碑记**。

---

## 八、工程档案管理（《管理办法》第二十五条）

- 业主、设计、施工、申报、审批机关应建立 **行政、技术、财务文件档案管理制度**；
- 工程资料 **立卷存档** 并归入 **文物保护单位记录档案**；
- 按《文物保护工程文件归档整理规范》整理；
- 重要工程验收后三年内发表技术报告。

## 九、实施阶段流程总图

```
┌────────────────────────────────────────────────────────────┐
│  招投标（修缮/保护设施/迁移）→ 选定施工、监理单位          │
│                         ↓                                   │
│  开工报告 → 技术监督部门启动监管                           │
│  施工前培训 + 设立警示标识 + 管理人员在岗备案               │
│                         ↓                                   │
│  施工实施：现场复核→补充勘察→工程洽商→施工记录→阶段验收   │
│  （构件/关键工序/隐蔽工程/样板 影像记录）                   │
│                         ↓                                   │
│  [异常] 发现新文物 → 保护现场 → 报原审批机关                │
│  [变更] 技术设计变更→备案；方案重要变更→原审批机关批准      │
│                         ↓                                   │
│  工程检查（事中监管）→ 问题整改 → 复查                      │
│                         ↓                                   │
│  竣工：自检→四方验评→申报机关初验→审批机关竣工验收          │
│                         ↓                                   │
│  竣工档案归档（记录档案）+ 工程报告出版（重点项目3年内）    │
│                         ↓                                   │
│  保修期（≥5年）跟踪                                         │
└────────────────────────────────────────────────────────────┘
```

## 十、对管理系统设计的启示

1. **开工报告与人员备案**：开工报告审批、管理人员在岗考勤。
2. **施工过程记录**：施工日志、隐蔽工程验收、影像记录的结构化采集与归档。
3. **设计变更分级审批**：强校验"洽商≠变更"，变更走审批、洽商走备案。
4. **监理模块**：监理日志、监理意见、隐蔽工程验收会签。
5. **检查整改闭环**：检查任务→问题清单→整改→复查。
6. **技术监督流程**：开工报告→关键节点→材料检测→工地检查→竣工审核→监管记录。
7. **验收流程引擎**：自检→四方验评→初验→竣工验收，结论独立性保障。
8. **竣工资料校验**：必备清单强制校验，对比照片自动比对。
9. **档案管理**：全生命周期档案归集，按归档规范分类。
10. **保修期管理**：到期提醒、保修期内问题跟踪。
