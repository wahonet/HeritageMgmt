# 嘉祥县文物保护背景说明

> 本文档为「嘉祥县文物保护工程管理系统」提供 **地域性文物资源背景**，帮助准确理解业务对象与保护重点。
> 📅 整理日期：2026-06-19

---

## 一、嘉祥县基本情况

| 项目 | 内容 |
|------|------|
| **行政区划** | 山东省济宁市嘉祥县 |
| **地理位置** | 山东西南部，济宁市西部 |
| **文化地位** | "中国汉碑半济宁，嘉祥汉画甲天下"——全国汉画像石集中出土地之一 |
| **历史名人** | **曾子（曾参）**——孔子弟子、宗圣，故里在嘉祥 |
| **文物特色** | 以 **汉代画像石（石刻）** 闻名，兼有古建筑、古遗址、碑刻等 |

## 二、核心文物资源（⭐系统重点关注）

### 1. 武氏墓群石刻（武氏祠）—— 镇县之宝

| 项目 | 内容 |
|------|------|
| **级别** | **第一批全国重点文物保护单位**（1961年国务院公布） |
| **位置** | 嘉祥县武翟山村 |
| **年代** | 东汉桓帝、灵帝时期（始建于东汉） |
| **性质** | 武氏家族墓地上的一组 **全石结构** 石刻建筑艺术珍品 |
| **地位** | **中国规模最大、保存最完整的汉画像石群**，被誉为研究中国汉代历史的"百科全书" |
| **藏品** | 武氏墓群石刻博物馆藏品 **422件/套**：原祠堂汉画像石46块、汉碑2通、清碑11通、石狮石阙各一对、碣1块；另藏嘉祥县出土画像石百余块 |
| **艺术价值** | 汉画像石雕刻精美、内容丰富，生动再现汉代社会生活、政治、风俗、信仰、艺术 |
| **保护工程特点** | **石刻类文物保护**——涉及石刻防风化、加固、拓印管理、陈列展示设施建设等 |

### 2. 曾庙（曾子庙）

| 项目 | 内容 |
|------|------|
| **性质** | 祭祀曾子（宗圣）的专庙 |
| **规模** | 占地面积约29808平方米 |
| **类型** | **古建筑类**文物保护单位 |
| **保护工程特点** | 古建筑修缮（木作、瓦石作、油漆彩画作），须遵守古建筑修缮规程与传统工艺 |

### 3. 其他文物资源
- 嘉祥县出土的 **汉画像石** 遍布全县，遗存众多；
- 历代 **碑刻**（汉碑、隋碑、清碑等）；
- 古遗址、古墓葬；
- 与曾子文化、儒家文化相关的历史遗存。

## 三、文物保护工程的特点与系统适配（⭐设计重点）

### 1. 石刻类文物保护（武氏祠类）
- **病害类型**：风化、断裂、剥落、生物侵蚀、水害等；
- **工程类型**：石刻加固、防风化处理、保护棚/保护性设施建设、拓印管理、陈列展示工程；
- **资质要求**：勘察设计/施工单位须具备 **"石窟寺和石刻"** 业务范围；
- **特殊要求**：严禁翻新古代石刻、彩塑；石刻拓印须严格管理；
- **预算定额**：适用《文物建筑保护工程预算定额（北方地区）》WW/T 0085-2017。

### 2. 古建筑类保护（曾庙类）
- **工程类型**：修缮工程（结构加固、局部复原）、保养维护；
- **资质要求**：勘察设计/施工单位须具备 **"古建筑"** 业务范围；
- **技术要求**：遵守《古建筑修缮项目施工规程》《文物建筑修缮工程操作规程》；
- **传统工艺**：木作、瓦石作、油漆作、彩画作等分部工程；
- **特别注意**：木结构古建筑落架须经省级文物部门事先同意。

### 3. 审批权限（嘉祥县适用）
| 文物级别 | 示例 | 申报机关 | 审批机关 |
|----------|------|----------|----------|
| 全国重点文保 | 武氏墓群石刻 | 山东省文物局 | **国家文物局** |
| 省级文保 | （曾庙等若为省保） | 济宁市/嘉祥县文物部门 | 山东省文物局 |
| 市县级文保 | 县内各级文保 | 嘉祥县文物部门 | 济宁市文物部门 |
| 未定级 | 县内未定级不可移动文物 | 嘉祥县文物部门 | 按省规定确定 |

### 4. 适用的法规层级（嘉祥县）
1. 《中华人民共和国文物保护法》（2024修订，2025.3.1施行）
2. 《文物保护法实施条例》
3. 《文物保护工程管理办法》《文物保发〔2025〕41号》等
4. 《长城保护条例》（涉齐长城时）
5. **《山东省文物保护条例》（2026修订，2026.3.1施行）** ⭐
6. 《山东省齐长城保护条例》《山东省红色文化保护传承条例》
7. 济宁市、嘉祥县配套文件

## 四、嘉祥县文物保护的司法与社会实践

- 嘉祥县人民法院积极探索 **文物保护司法新路径**，体现了文物依法保护的地方实践。
- 山东曾子研究会、嘉祥汉画学会等学术团体活跃，推动曾子文化与汉画研究。
- 武氏墓群石刻博物馆是 **国务院首批公布的全国重点文物保护单位**，社会关注度高，相关工程易被列为 **重点项目**。

---

## 五、对管理系统设计的启示（⭐总结）

1. **文物对象分类**：支持 **石刻类、古建筑类、古遗址古墓葬类、碑刻类** 等嘉祥县主要文物类型。
2. **资质业务范围校验**：石刻工程匹配"石窟寺和石刻"资质，古建筑匹配"古建筑"资质。
3. **审批路由**：嘉祥县境内国保（武氏祠）工程直报国家文物局，系统须支持跨级申报。
4. **重点项目标记**：武氏祠类高关注度工程标记为重点项目，强化资金与报告出版管理。
5. **石刻专项管理**：拓印管理、防风化监测、保护设施建设等专项功能。
6. **省级平台对接**：预留与山东省"全省统一文物资源数据信息平台"对接接口。
7. **多法规适用**：系统应能识别文物适用的全部法规层级，提示合规要点。
8. **地域文化特色**：融入曾子文化、汉画文化等地方文化元素的展示与利用管理。
