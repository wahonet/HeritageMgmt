# 文物保护工程立项与勘察设计流程

> 本文档梳理文物保护工程 **前期阶段（立项 → 勘察 → 方案设计 → 施工技术设计）** 的完整流程与审批权限。
> 依据：《文物保护工程管理办法》第十～十六条、《文物保发〔2025〕41号》第一部分。

---

## 一、分级报批权限总表（⭐核心）

依据《文物保护工程管理办法》第十条：

| 文物保护单位级别 | 申报机关 | 审批机关 |
|------------------|----------|----------|
| **全国重点文物保护单位** | 省/自治区/直辖市文物行政部门 | **国家文物局** |
| **省级文物保护单位** | 文物所在地的市/县级文物行政部门 | 省/自治区/直辖市文物行政部门 |
| **市县级文物保护单位** | 由省级文物行政部门确定申报/审批机关 | （通常为市级文物行政部门） |
| **未定级不可移动文物** | 由省级文物行政部门确定 | （通常为县级文物行政部门） |

> 📌 **山东/嘉祥县适用**：
> - 嘉祥县境内若有 **国保**（如武氏墓群石刻），其修缮工程方案由山东省文物局申报、国家文物局审批。
> - **省保** 由济宁市/嘉祥县文物部门申报、山东省文物局审批。
> - **市保/县保/未定级** 由济宁市文物部门确定，通常嘉祥县文物部门参与。

## 二、立项阶段

### 1. 立项前置
- 文物保护单位应制定 **专项总体保护规划**，工程应依据批准的规划进行。
- 业主单位（具有法人资格的文物管理或使用单位）作为立项主体。
- 2025年41号通知要求：对 **保存状况不佳、多年未修缮** 的文物建筑主动核查、督促修缮。

### 2. 立项申报资料（《管理办法》第十三条，⭐系统表单字段）

| 序号 | 申报资料内容 | 系统字段 |
|------|--------------|----------|
| 1 | 工程业主单位及上级主管部门名称 | 业主单位、主管部门 |
| 2 | 拟立项目名称、地点，文物保护单位级别、时代 | 项目名称、地点、级别、时代 |
| 3 | 保护范围与建设控制地带的划定、公布与执行情况 | "两线"信息 |
| 4 | 保护工程必要性与实施可能性的技术文件与形象资料（录像/照片） | 技术文件、照片/视频附件 |
| 5 | 经费估算、来源及计划工期安排 | 经费估算、资金来源、工期 |
| 6 | 拟聘请的勘察设计单位名称及资信 | 勘察设计单位、资质信息 |

### 3. 不同工程类型的立项程序差异
- **保养维护工程**：列入年度计划，**报省级文物部门备案**（无需立项审批）。
- **抢险加固工程**：按分级报批；紧急可 **边实施边补报**。
- **修缮/保护性设施建设工程**：按分级报批立项与勘察设计方案。
- **迁移工程**：先依《文物保护法》获 **迁移批准**（国保须国务院批准），再报批勘察设计方案。
- **重建工程**：先依《文物保护法》获批准，再报批勘察设计方案。

## 三、勘察设计阶段

### 1. 设计文件分阶段编制（《管理办法》第十四～十六条）

```
勘察 → 方案设计 → [重大工程：技术设计] → 施工技术设计（施工图设计）
```

> 📌 **重大工程** 在方案获批准后，再进行技术设计。

### 2. 勘察和方案设计文件（必备内容）

| 类别 | 内容 |
|------|------|
| 勘察成果 | 反映文物历史状况、固有特征和损害情况的 **勘察报告、实测图、照片** |
| 方案 | 保护工程 **方案、设计图** 及相关技术文件 |
| 概算 | **工程设计概算** |
| 附加（必要时） | 考古勘探发掘资料、材料试验报告书、环境污染报告书、工程地质与水文地质资料 |

### 3. 施工技术设计文件（必备内容）

| 类别 | 内容 |
|------|------|
| 图纸 | **施工图** |
| 说明 | **设计说明书** |
| 预算 | **施工图预算** |
| 检测 | 相关 **材料试验报告** 及 **检测鉴定结果** |

### 4. 设计深度要求
- 依据《文物保护工程设计文件编制深度要求（试行）》（办保函〔2013〕375号）。
- 由 **责任设计师** 统筹，对粉刷、勾缝、油饰、彩画等新旧并存情况牵头提出技术措施。

---

## 四、方案设计理念要求（2025年41号通知，⭐审查要点）

- 严格按照《设计文件编制深度要求》和立项批复意见编制方案；
- **原则上不以恢复至某一特定历史时期为修缮目标**；
- 甄别、评估、保存各时期有价值、有时代特征的构件、形制和历史信息；
- 没有油饰彩画传统的地区或油饰彩画已基本缺失的建筑群，一般不恢复；
- **1911年以前的彩画、灰塑原则上现状科技保护，严禁翻新古代壁画和彩塑**；
- 保存墙体历史沧桑感，不宜整体粉刷古塔；
- 新旧总体协调、兼顾可识别性。

---

## 五、审批从严要求（2025年41号通知）

- 严格落实 **不改变文物原状** 和 **最小干预** 原则；
- 谨慎、细致评审审批，**严禁超范围审批**；
- 鼓励依托专业技术单位开展技术评审；
- **市县级文物部门批复木结构古建筑落架方案的，须事先征求省级文物部门意见；其他类型文物建筑不得落架**。

---

## 六、重点项目管理

国家文物局加大重点项目比例：
- 世界文化遗产、壁画彩塑；
- 涉及国家重大战略、社会广泛关注、技术难度大的工程；
- 加大资金保障，鼓励先行先试；
- **重点项目应在工程竣工验收后编辑出版文物保护工程报告**。

│  2. 业主单位提出工程需求（主动核查/日常排查）                │

│  4. 编制立项申报资料 → 报相应级别申报机关 → 审批机关立项批复 │

│                         ↓                                    │

## 八、对管理系统设计的启示

1. **立项表单**：严格按第十三条6项字段建模，支持附件上传。
2. **分级审批路由引擎**：根据文物级别自动匹配申报机关与审批机关。
3. **设计文件分阶段管理**：勘察/方案/技术设计三阶段分别建档、版本控制、独立审批。
4. **设计单位资质校验**：选择勘察设计单位时校验资质等级是否覆盖工程级别。
5. **设计深度校验清单**：按《深度要求》设置必备文件清单校验。
6. **理念合规自检**：方案提交时强制填报原则落实情况。
7. **重点项目标记**：重点项目单独管理流程与资金、报告出版跟踪。
