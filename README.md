## 三、近两年最关键的变化（务必重点关注）

1. **《中华人民共和国文物保护法》2024年大修**
   - 2024年11月8日十四届全国人大常委会第十二次会议修订通过，**自2025年3月1日起施行**。
   - 由原 80 条增至 **101 条**（新增19条、修改75条），首次在法律中明确"文物"定义，大幅提高罚款上限（情节严重者单位最高 **1000万元**，原为50万元）。

2. **《山东省文物保护条例》2026年大修**
   - 2026年1月25日山东省第十四届人大常委会第二十次会议修订通过，**自2026年3月1日起施行**。
   - 共 **8章70条**，新增"研究与利用""保障与监督"两章，强化"先考古、后出让""先调查、后建设"前置制度。

3. **《国家文物局关于加强文物建筑修缮工程全流程管理的通知》（文物保发〔2025〕41号）**
   - 2025年9月30日发布并施行，是当前文物建筑修缮工程 **最新的全过程管理要求**，
   - 推动建立 **技术监督体系**、统一全流程管理平台。

## 四、使用建议

- **系统建模**：参照 [3-Process/](3-Process/) 中的工程分类与全流程文档进行数据模型与审批流程设计。
- **权限设计**：参照 [1-National/05_文物保护工程资质管理办法.md](1-National/05_文物保护工程资质管理办法.md) 及 [3-Process/04_资质资格管理体系.md](3-Process/04_资质资格管理体系.md) 设计用户角色与资质分级。
- **审批流程**：参照 [3-Process/02_立项与勘察设计流程.md](3-Process/02_立项与勘察设计流程.md) 与 [3-Process/03_施工监理与验收流程.md](3-Process/03_施工监理与验收流程.md) 设计立项→勘察设计→施工→监理→验收的审批节点。
- **合规校验**：在系统中内化"不改变文物原状""最小干预""先考古后出让"等法定原则。
